# Décisions Durables

Ce fichier garde les décisions stables qui aident les agents à éviter la redécouverte. Il ne sert pas de journal de tâches.

## 2026-04-27 - Mémoire Agent Progressive

- `AGENTS.md` reste le point d'entrée normatif et court.
- Les détails opérationnels vivent dans `docs/agent-context/`.
- Les workflows répétitifs vivent dans des skills versionnés sous `.codex/skills/`.
- Si une fiche mémoire contredit le code actuel, le code gagne.
- Les fiches doivent être mises à jour seulement pour des conventions durables, pièges réels ou décisions architecture/produit.
